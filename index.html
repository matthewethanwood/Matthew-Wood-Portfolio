This game was made by me and me alone. I had worked on this project over the course of one year, which involved researching the tools that I needed in order to make this application. This is one version of the game in the video, the other version is identical in visuals; the only difference between the two is the actual adaptive difficluty algorithm that was used for each of them. The one in the video uses item response theory (IRT), which is a pschoanalytical test that was used to determine the likelyhood of a person answering a question correctly. The other version of this game uses a neural network based algorithm (NNA), which is an AI model that was trained to change certain values in the code to adjust the difficulty appropriately to the user. The program also records the answers of the player as well as other important details such as which questions that they answered, because each person should not receive the same set of questions in the same order, how long it took them to answer the question, of course, whether they are right or wrong, and their final score at the end.
